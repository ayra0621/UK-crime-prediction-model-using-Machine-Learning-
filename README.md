# UK-Crime-Prediction-Model-Using-Machine-Learning-
This project was developed as part of my MSc Data Science dissertation at Leeds Beckett University.

The objective is to forecast crime trends across England and Wales and analyse borough-level crime patterns in London using machine learning models and socio-economic indicators.

## Project Structure
1. England & Wales Model  
Quarterly crime prediction across Police Force Areas using Random Forest.
2. London Borough Model  
Monthly crime prediction using XGBoost with socio-economic indicators from the Index of Multiple Deprivation (IMD).
3. Streamlit Dashboard  
Interactive dashboard to visualise predictions and risk labels (High / Medium / Low).

## Technologies Used
- Python  
- pandas  
- scikit-learn  
- XGBoost  
- Streamlit  
- matplotlib  

## Data Sources
- UK Police Data Portal  
- Office for National Statistics (ONS)  
- English Indices of Deprivation (IMD)
